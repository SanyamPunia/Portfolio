# ✨ sanyam.xyz

* [Next.js](https://nextjs.org/) - Framework
* [Sanity.io](https://www.sanity.io/) - Content Management System (CMS) for blog posts
* [Tailwind CSS](https://tailwindcss.com/) - CSS Framework
* [VERCEL](https://vercel.com/) - Deployment

# 🔬 Local Setup
```js
npm install
npm run dev
```
and add the required `env` variables by creating `.env.local` file at the root directoy
```
NEXT_PUBLIC_SANITY_DATASET=
NEXT_PUBLIC_SANITY_PROJECT_ID=
```
